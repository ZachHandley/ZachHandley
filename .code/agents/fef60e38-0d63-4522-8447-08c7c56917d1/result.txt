# 🚀 Threlte/Three.js Performance Optimization Plan

## Executive Summary
Your 3D portfolio site (1.74MB gzipped LinkApp chunk) can be optimized through incremental, low-risk improvements. Focus areas: asset optimization, renderer settings, Threlte-specific patterns, and spatial layout refinements.

## 📊 Current State Analysis

### Strengths
- ✅ Draco compression already implemented
- ✅ Camera update throttling (200ms)
- ✅ Particle pool pre-warming system
- ✅ Responsive crate positioning logic
- ✅ Svelte 5 runes for reactivity

### Performance Bottlenecks
- ❌ No texture compression (81KB cloud.png unoptimized)
- ❌ Large CrateExplode model (277KB)
- ❌ No pixel ratio clamping
- ❌ Missing LOD/instancing for repeated crates

// sceneController.svelte.ts
class SceneController {
  cullInvisibleCrates(camera: THREE.Camera) {
    const frustum = new THREE.Frustum();
    frustum.setFromProjectionMatrix(
      new THREE.Matrix4().multiplyMatrices(
        camera.projectionMatrix,
        camera.matrixWorldInverse
      )
    );
    
    this.crateRegistry.forEach((crate, id) => {
      const visible = frustum.containsPoint(crate.position);
      crate.component.setVisible(visible);
    });
  }
}

## 📐 Phase 4: Crate Placement Improvements
**Timeline: 1-2 days | Risk: Low | Impact: Medium**

### 4.1 Improved Spatial Algorithm
```typescript
// crateController.svelte.ts improvements
class CrateController {
  calculateOptimalLayout(links: Link[], viewport: ViewportDimensions) {
    const isMobile = viewport.width < 768;
    
    // Adaptive density based on device
    const CRATE_SPACING_FACTOR = isMobile ? 1.2 : 1.5;
    const MAX_CRATES_PER_COLUMN = isMobile ? 3 : 4;
    
    // Avoid dragon occlusion with better margins
    const DRAGON_SAFE_ZONE = {
      x: this.options.dragonWidth * 1.3, // 30% margin
      y: 5 // Height clearance
    };
    
    // Spiral placement for better distribution
    const positions = this.calculateSpiralLayout(
      links.length,
      DRAGON_SAFE_ZONE,
      CRATE_SPACING_FACTOR
    );
    
    return positions;
  }
  
  private calculateSpiralLayout(
    count: number,
    safeZone: { x: number, y: number },
    spacing: number
  ): [number, number, number][] {
    const positions: [number, number, number][] = [];
    let angle = 0;
    let radius = safeZone.x;
    
    for (let i = 0; i < count; i++) {
      const x = Math.cos(angle) * radius;
      const y = (i / count) * safeZone.y;
      positions.push([x, y, this.options.zDepth]);
      
      angle += (Math.PI * 2) / 8; // 8 positions per circle
      if (i % 8 === 7) radius += spacing;
    }
    
    return positions;
  }
}
```

## 📊 Metrics & Monitoring

### Key Performance Indicators
```typescript
// utils/performanceMonitor.ts
class PerformanceMonitor {
  private stats = {
    fps: 0,
    frameTime: 0,
    memoryUsage: 0,
    drawCalls: 0
  };
  
  measure() {
    // Using three.js renderer.info
    this.stats.drawCalls = renderer.info.render.calls;
    this.stats.memoryUsage = renderer.info.memory.geometries;
    
    // Custom FPS counter
    const delta = clock.getDelta();
    this.stats.fps = 1 / delta;
    this.stats.frameTime = delta * 1000;
    
    // Send to analytics
    if (this.stats.fps < 30) {
      console.warn('Low FPS detected:', this.stats);
    }
  }
}
```

<script lang="ts">
  import { useTask, useThrelte } from '@threlte/core';
  import { Suspense } from '@threlte/extras';
  
  const { renderer, invalidate } = useThrelte();
  
  // Clamp pixel ratio
  $effect(() => {
    renderer.setPixelRatio(Math.min(2, window.devicePixelRatio));
  });
  
  // Pause rendering when hidden
  document.addEventListener('visibilitychange', () => {
    if (document.hidden) {
      renderer.setAnimationLoop(null);
    } else {
      invalidate();
    }
  });
